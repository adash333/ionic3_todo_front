詳細は以下
http://twosquirrel.mints.ne.jp/?p=18681

## Requirement

- Rails 5
- Node.js
- Ionic CLI

## Installation

    $ git clone git@github.com:adash333/ionic3_todo_front.git
    $ cd ionic3_todo_front.git
    $ npm install @ionic/app-scripts@latest --save-dev
    $ npm install
    $ ionic serve
